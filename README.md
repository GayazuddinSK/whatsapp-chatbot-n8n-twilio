# WhatsApp Automation Bot (n8n + Twilio)

##  Overview

This project is a WhatsApp chatbot built using n8n and Twilio API.

It can:

* Auto reply to messages
* Show menu options (Courses, Fees, Contact)
* Store user data in Google Sheets
* Handle multiple user inputs

##  Tech Stack

* n8n (Workflow Automation)
* Twilio WhatsApp API
* Google Sheets

##  Workflow

1. User sends message on WhatsApp
2. Webhook receives message
3. Data saved in Google Sheets
4. IF logic processes message
5. Bot sends reply via Twilio API
